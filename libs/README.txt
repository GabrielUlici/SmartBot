The following libraries are required to successfully build:

android-support-v4.jar
  - The Android Support Library (r8 or later)
    Install from the Android SDK Manager.
joystickview.jar
  - Android JoystickView is a Custom View that
  	simulates a Joystick for interactive
  	applications for Android, as simple aim is
  	allows access values ​​of angle and power of
  	the virtual Joystick movement.
  	
    See https://github.com/zerokol/JoystickView
stanford-parser.jar
  - Implementations of probabilistic natural language parsers
    See http://nlp.stanford.edu/software/lex-parser.shtml
